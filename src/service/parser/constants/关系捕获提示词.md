除此之外只标注from、to。缺少from的需要在现有查询基础上包装高层模式，例如)。to为调用等关系指向的对象，from为关系发起的实体。

(call_expression
  function: (identifier) @call.function.name
  arguments: (argument_list
    (_) @call.argument)*) @relationship.call 
改造为
(function_definition
  declarator: (function_declarator
    declarator: (identifier) @relationship.call.from) ;
  body: (compound_statement
    (_
      (call_expression
        function: (identifier) @relationship.call.to
      ) @relationship.call ; 将类型捕获放在这里
    )
  )
)
