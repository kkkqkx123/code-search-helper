## 5. 数组声明查询

### 测试用例
```c
// 数组声明测试
int numbers[10];
char buffer[256];
float matrix[5][5];
double values[20];
char text[100] = "Hello";
int dynamic_array[];
```

### 查询规则
```
(declaration
  type: (_)
  declarator: (array_declarator
    declarator: (identifier) @array.name
    size: (_)? @array.size)) @definition.array
```
