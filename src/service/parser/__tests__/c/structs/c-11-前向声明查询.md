## 11. 前向声明查询

### 测试用例
```c
// 前向声明测试
struct Node;  // 结构体前向声明
union Storage;  // 联合体前向声明
enum State;    // 枚举前向声明

// 完整定义
struct Node {
    int data;
    struct Node* next;
};

union Storage {
    int i;
    float f;
    char c;
};

enum State {
    INIT,
    RUNNING,
    STOPPED
};

int main() {
    struct Node* head = NULL;
    return 0;
}
```

### 查询规则
```
(struct_specifier
  name: (type_identifier) @forward.struct.name) @definition.forward.struct

(union_specifier
  name: (type_identifier) @forward.union.name) @definition.forward.union

(enum_specifier
  name: (type_identifier) @forward.enum.name) @definition.forward.enum
```
