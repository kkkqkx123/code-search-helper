## 4. 类型别名查询

### 测试用例
```c
// 类型别名定义测试
typedef int Integer;
typedef char* String;
typedef struct Point Point;
typedef enum Color Color;
typedef union Data Data;

// 函数指针类型别名
typedef int (*Comparator)(int a, int b);
```

### 查询规则
```
(type_definition
 type: (_)
  declarator: (type_identifier) @alias.name) @definition.type.alias
```
