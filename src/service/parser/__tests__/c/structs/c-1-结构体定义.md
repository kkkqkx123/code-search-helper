## 1. 结构体定义查询

### 测试用例
```c
// 结构体定义测试
struct Point {
    int x;
    int y;
};

struct Person {
    char name[50];
    int age;
    float height;
};

// 嵌套结构体
struct Rectangle {
    struct Point top_left;
    struct Point bottom_right;
};
```

### 查询规则
```
(struct_specifier
  name: (type_identifier) @type.name
  body: (field_declaration_list
    (field_declaration
      type: (_) @field.type
      declarator: (field_identifier) @field.name)*)) @definition.struct
```
