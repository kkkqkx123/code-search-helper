## 15. 条件变量销毁同步关系

### 测试用例
```c
#include <pthread.h>

int main() {
    pthread_cond_t cond;
    
    pthread_cond_init(&cond, NULL);
    
    // pthread_cond_destroy测试
    pthread_cond_destroy(&cond);
    
    return 0;
}

```
