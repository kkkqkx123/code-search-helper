## 25. 信号量销毁同步关系

### 测试用例
```c
#include <semaphore.h>

int main() {
    sem_t sem;
    
    sem_init(&sem, 0, 1);
    sem_wait(&sem);
    sem_post(&sem);
    
    // sem_destroy测试
    sem_destroy(&sem);
    
    return 0;
}

```

### 查询规则
```

  (#match? @semaphore.destroy.function "^(sem_destroy)$")

    (pointer_expression argument: (identifier) @semaphore.handle))

```
