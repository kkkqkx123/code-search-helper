## 快速开始

如果您想快速了解图索引功能的实现，建议直接阅读 [实现指南](./implementation-guide.md)，该文档包含了所有必要的信息和代码示例。

如果您想了解特定方面的详细设计，可以参考相应的专门文档。

## 实现优先级

1. **高优先级**: 阅读 [实现指南](./implementation-guide.md) 并按照其中的步骤进行实现
2. **中优先级**: 参考 [架构设计](./index-design.md) 了解系统整体架构
3. **低优先级**: 参考 [查询规则设计](./rule-design.md) 和 [代码设计](./code-design.md) 了解具体实现细节

## 注意事项

- 所有文档都已根据现有项目结构进行了调整
- 实现方案充分利用了现有的服务和组件
- 新增的组件最小化，主要是 `GraphMapperService`
- 查询规则基于现有的 `QueryTypeMappings.ts` 和查询模式进行了扩展
